FAQ

Where do you obtain the content for each book?
Each individual word of each novel is obtained through Project Gutenberg and Goodreads. All of the novels currently on the site are on the public domain and outside of copyright infringement. All of the files contain original code and Fandom Nerds did not copy and use any files from Project Gutenberg or Goodreads.
What is the public domain?
The public domain consists of novels that have been published for 70 years or more and, therefore, are outside of copyright law. Any one can use the words and text of each public domain novel so long as the individual or corporation is not copying the code and files of where they obtained a novel on the public domain.
Do I have to sign up to access authors and/or the ebooks on this site?
No, you do not need to sign up and create an account with Fandom Nerds in order to read the public domain ebooks on this site. However, Fandom Nerds does encourage you to sign up to receive updates on when new novels are available on this site as well as any updates or new features to the site itself.
